# Кодекс поведения участников

## Наши обязательства

Мы стремимся создать открытую и дружелюбную среду для всех участников проекта CRM.

## Ожидаемое поведение

- Уважительное общение в issues и pull request'ах.
- Конструктивная критика, сфокусированная на коде и задаче.
- Готовность помогать новым участникам.

## Недопустимое поведение

- Оскорбления, дискриминация, травля.
- Публикация чужих персональных данных без согласия.
- Саботаж обсуждений и злоупотребление доступом к репозиторию.

## Сообщить о нарушении

Напишите maintainers репозитория через Issues с пометкой «поведение» или свяжитесь с владельцем организации Corpuscul на GitVerse.

## Адаптация

Документ основан на [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), версия 2.1.
